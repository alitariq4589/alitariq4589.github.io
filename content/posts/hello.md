This is a starter post showing the file format. Each post is a markdown file in
content/posts/. The block at the top (between the +++ lines) is front matter:
title, date, tags, and a one-line summary that shows on the post list.

To publish the ci-medic launch post, drop its markdown file in this folder, set
draft = false, and push. The GitHub Action builds and deploys automatically.
